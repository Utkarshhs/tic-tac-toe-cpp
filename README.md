# Tic Tac Toe Game

A simple Tic Tac Toe game built using C++ that runs in the console.

##  Features

* Two-player mode
* Clear and interactive board display
* Detects winners and draw situations
* Validates moves and prevents invalid positions

##  Technologies Used

* C++
* Standard Template Library (STL)

##  How to Run

1. Clone the repository:

```bash
git clone https://github.com/Utkarshhs/tic-tac-toe.git
```

2. Navigate into the project folder:

```bash
cd tic-tac-toe
```

3. Compile the source code:

```bash
g++ main.cpp -o tictactoe
```

4. Run the executable:

```bash
./tictactoe
```
